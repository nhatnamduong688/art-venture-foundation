# Footer Images

This folder contains images and SVG files for the Footer section.

## Files needed:
- Logo (SVG preferred for sharp display)
- Social media icons (if any)
- Payment method icons (if any)
- Any other footer graphics

## Instructions:
1. Upload your images/SVG files to this folder
2. Use descriptive names (e.g., `logo.svg`, `facebook-icon.svg`, etc.)
3. For logos, SVG format is recommended for best quality
4. For images, use optimized formats (WebP, PNG, or JPEG)

## Image Guidelines:
- **Logos**: SVG format, transparent background
- **Icons**: SVG or PNG with transparency
- **Photos**: WebP or JPEG, optimized for web

Once uploaded, notify me and I'll integrate them into the Footer component.
