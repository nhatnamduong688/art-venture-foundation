# Event Detail Page Images

This folder contains images for the Event Detail page (`/events/:id`).

## Image Structure

Each event detail page has **2 images**:
1. **Hero Image** - Large banner image at the top
2. **Content Image** - Image in the middle of the article

## Image Requirements

### Hero Image:
- **Format**: JPG or WebP
- **Dimensions**: ~1200-1600px width (large, landscape)
- **Aspect Ratio**: Wide (16:9 or 2:1)
- **Naming**: `hero-1.jpg` (for event 1), `hero-2.jpg`, `hero-3.jpg`

### Content Image:
- **Format**: JPG or WebP
- **Dimensions**: ~800-1000px width (medium, landscape)
- **Aspect Ratio**: Wide or square
- **Naming**: `content-1.jpg` (for event 1), `content-2.jpg`, `content-3.jpg`

## How to Add Images

1. Export hero and content images from Figma for event 1
2. Save them as `hero-1.jpg` and `content-1.jpg`
3. Repeat for other events if needed (hero-2, hero-3, etc.)
4. I'll update EventDetailPage to reference these local paths
