# Artist Avatar Images

This folder contains avatar images for artists displayed in the Collection page.

## Image Requirements

### Recommended Specs:
- **Format**: JPG or PNG
- **Dimensions**: Square (e.g., 200x200px, 300x300px)
- **Quality**: 80-90% for good balance
- **File naming**: `artist-name.jpg` or generic like `artist-1.jpg`

## How to Add Images

1. Export artist avatar from Figma
2. Save to this folder
3. Update the component to reference the local path
4. Avatar will be displayed in the Collection page artist info section
