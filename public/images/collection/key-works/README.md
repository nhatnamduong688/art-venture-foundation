# Key Works Grid Images

This folder contains artwork images for the "Key works" grid section on the Collection page (`/collection`).

## Current Grid Structure

The grid has **8 artworks** with different sizes:
- **1 large** (id: 1)
- **6 medium** (id: 2, 3, 5, 6, 7, 8)
- **1 small** (id: 4)

## Categories
- **People** (portraits)
- **Nature** (landscapes)
- **Sculpture** (3D art)

## Image Requirements

### Recommended Specs:
- **Format**: JPG or WebP
- **Quality**: 80-90% for good balance
- **Dimensions**: Varies by grid size
  - Large: ~800-1000px width
  - Medium: ~500-700px width
  - Small: ~300-400px width

### File Naming:
- `artwork-1.jpg` (large - nature)
- `artwork-2.jpg` (medium - sculpture)
- `artwork-3.jpg` (medium - people)
- `artwork-4.jpg` (small - people)
- `artwork-5.jpg` (medium - sculpture)
- `artwork-6.jpg` (medium - nature)
- `artwork-7.jpg` (medium - people)
- `artwork-8.jpg` (medium - sculpture)

## How to Add Images

1. Export 8 artwork images from Figma
2. Save them to this folder with names `artwork-1.jpg` through `artwork-8.jpg`
3. I'll update the component to reference these local paths
