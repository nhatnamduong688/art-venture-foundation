# Knowledge Page Images

This folder contains images for the Knowledge page categories.

## Structure

The Knowledge page has **5 categories**, each with:
1. **Icon Image** - Small icon in the header
2. **Hero Image** - Large banner image

### Categories:
1. **Arts Museums** (`museums`)
2. **Art Schools** (`schools`)
3. **Art Books** (`books`)
4. **Art Magazines** (`magazines`)
5. **Art Spaces & Galleries** (`spaces`)

## Image Requirements

### Icons (`/icons/`):
- **Format**: PNG or SVG (preferred for icons)
- **Dimensions**: ~100-150px (small icons)
- **Background**: Transparent preferred
- **Naming**: `museums.png`, `schools.png`, `books.png`, `magazines.png`, `spaces.png`

### Hero Images (`/heroes/`):
- **Format**: JPG or WebP
- **Dimensions**: ~800-1200px width (landscape)
- **Aspect Ratio**: Wide (16:9 or similar)
- **Naming**: `museums.jpg`, `schools.jpg`, `books.jpg`, `magazines.jpg`, `spaces.jpg`

## How to Add Images

1. Export 5 icon images and 5 hero images from Figma
2. Save icons to `/icons/` folder
3. Save hero images to `/heroes/` folder
4. I'll update KnowledgePage to reference these local paths
