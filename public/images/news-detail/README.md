# News Detail Page Images

This folder contains images for the News Detail page (`/news/:id`).

## Image Structure

Each news detail page has **2 images**:
1. **Hero Image** - Large banner image at the top
2. **Content Image** - Image in the middle of the article

## Image Requirements

### Hero Image:
- **Format**: JPG or WebP
- **Dimensions**: ~1200-1600px width (large, landscape)
- **Aspect Ratio**: Wide (16:9 or 2:1)
- **Naming**: `hero-1.jpg` (for news 1), `hero-2.jpg`, etc.

### Content Image:
- **Format**: JPG or WebP
- **Dimensions**: ~800-1000px width (medium, landscape)
- **Aspect Ratio**: Wide or square
- **Naming**: `content-1.jpg` (for news 1), `content-2.jpg`, etc.

## How to Add Images

1. Export hero and content images from Figma for news 1
2. Save them as `hero-1.jpg` and `content-1.jpg`
3. Repeat for other news articles if needed
4. I'll update NewsDetailPage to reference these local paths
